#### **项目描述：**

随着智能手机的普及和摄影技术的进步，越来越多的用户开始关注照片编辑和美化功能，促使了图片编辑类项目的发展。本项目包含多种功能，如添加水印，辅助相机，绘画画板，自由拼图等，功能丰富，界面美观，操作便捷，通过小程序的形式，用户无需下载独立的App，即可方便地使用相关图片编辑的功能，提高用户体验和使用便捷性。

#### **系统运行：**

使用微信开发者工具，扫码即可登录，即可编译并运行项目
